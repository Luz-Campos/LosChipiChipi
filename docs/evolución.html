1.1 EVOLUCIÓN DE LAS APLICACIONES WEB

Inicio 1.1 Evolución de las aplicaciones web. 1.2 Arquitectura de las aplicaciones web. 1.3 Tecnologías para el desarrollo de aplicaciones web. 1.4 Planificación de aplicaciones web.

5 de diciembre de 1969 - Arpanet

Formulada por Joseph CR Licklider de BBN y la ARPA, fue la forma inicial de internet y se creó con multas militares.

1980 - HTML

1980 el físico Tim Berners lee propone un nuevo sistema de hipertextos para compartir documentos.

1983 - C++

por Bjarne Stroustrup, la intención de su creación fue el extensor al lenguaje de programación C mecanismos que permiten la manipulación de objetos.

1986 - PostgreSQL

Empezó como un proyecto en la universidad de California en Berkeley a manos del profesor Michael Stonebraker.

1987 - Creación de "Perl"

Fue inventado por Larry Wall, "Perl" es uno de los primeros lenguajes para el desarrollo de aplicaciones web.

1989 - Servidor SQL

Microsoft se unió a Ashton-Tate y Sybase para crear una variante de Sybase SQL Server para IBM OS/2.

1991 - Red 1.0

Es la forma más básica que existe, con navegadores de sólo texto bastante rápidos ya que es de sólo lectura. El usuario no puede interactuar con el contenido de la página (nada de comentarios, respuestas, citas, etc), estando totalmente limitado a lo que el Webmaster sube a ésta.

1991 - PITÓN

Creado por Guido Van Rossum, este lenguaje de programación fue creado con la intención de ser un lenguaje sencillo de usar e interpretar.

1991 - WWW

El concepto de WWW fue diseñado por Tim Berners-Lee y algunos científicos del CERN (Conseil Européen pour la Recherche Nucléaire) en Ginebra.

1994 - Ópera Navegador

Nació como un proyecto de investigación en Telenor, la compañía más grande de telecomunicaciones Noruega.

15 de diciembre de 1994 - Netscape

Es un navegador web, el primer producto comercial de la compañía Netscape Communications creado por Marc Andreessen.

1995 - Creación de PHP

El programador Rasmus Lerdorf puso a disposición el lenguaje PHP lo que dio apertura al desarrollo de aplicaciones web.

1995 - Anuncio de la creación de JavaScript

Unos meses después del lanzamiento de PHP, Netscape, el navegador web más antiguo y popular, anunció una nueva tecnología, JavaScript, lo que permitiría a los programadores cambiar de forma dinámica el contenido de una página web.

1995 - Rubí

Es un lenguaje de programación interpretado, reflexivo y orientado a objetos, creado por el programador japonés Yukihiro "Matz" Matsumoto.

1995 - Java

Es un lenguaje de programación y una plataforma informática comercializada por primera vez en 1995 por Sun MycroSystems.

1995 - Explorador de Internet

¡Internet Explorer fue lanzado por primera vez para su instalación como un complemento de Microsoft Plus! para Windows 95

1996 - Anuncio de la creación de un servicio de correo

Dos desarrolladores, Sabeer Bhatia y Jack Smith lanzaron Hotmail, un servicio de correo en línea que permitiría por primera vez acceder y consultar el correo electrónico para el público en general.

1996 - Yahoo!

Fundado por Jerry Yang y David Filo, Yahoo es una empresa de software que creó su propio buscador de internet.

1996 - Microsoft IIS

Es una serie de servicios para los ordenadores que funcionan con Windows. Originalmente era parte del Option Pack para Windows NT

1997 - Destello

En 1997, la Plataforma ShockWave Flash hace su aparición por primera vez, la cual fue más tarde comprada por Adobe y Macromedia.

17 de enero de 1998 - Primer sitio web de noticias

El 17 de enero de 1998, el sitio web The Drudge Report anunció por primera vez un informe de noticias antes de que se difundiera en los medios de televisión y la prensa tradicional, esto causa revuelo para los medios de comunicación impresos y dio apertura a los medios de comunicación en línea.

15 de enero de 1998 - Motor de búsqueda de Google

Este mismo año, la compañía Google anunciaría su primer motor de búsqueda en línea que, por su nueva forma de indexar páginas web, facilitará enormemente la búsqueda de información en internet.

1999 - Gato apache

Apache Tomcat es desarrollado, en un entorno abierto y participativo y publicado bajo la licencia Apache versión 2, desarrollado por James Duncan Davidson.

2000 - Ópera 4.0

Incluía un nuevo núcleo de plataformas cruzadas que facilitaba la creación de las ediciones de Opera de múltiples sistemas operativos y plataformas, y también incluía la navegación por pestañas

2001 - Wikipedia

A Inicios de 2001, Wikipedia se anunciaría como un subproyecto de Nupedia, lo cual consistía básicamente en una enciclopedia en línea, hoy en día cuenta con casi 21 millones de artículos en distintos idiomas.

13 de febrero 2002 - Marco .NET 1.0

Liberado para Win98, Me, NT 4.0, 2000 y XP. Su soporte terminó en julio de 2007 y el soporte extendido en 2009.

2003 - Fundación de MySpace

En 2003 fue fundado MySpace para posteriormente convertirse en la red social más visitada, fue plataforma de lanzamiento para otros sitios como Youtube y Slide.com.

3 de abril de 2003 - Firefox

El proyecto Firefox comenzó como una rama experimental de Mozilla a cargo de Dave Hyatt, Joe Hewitt y Blake Ross, quienes consideraron que las exigencias comerciales del patrocinio de Netscape y el gran número de características de Mozilla Application Suite, comprometían la utilidad de este. Decidieron entonces crear un navegador independiente.

2004 - web 2.0

En una conferencia del Web 2.0, el concepto de "web como plataforma" fue mencionado por primera vez. Esta innovación allanó el camino para futuras aplicaciones web, es decir, un software que aprovecha las ventajas de la conexión a internet y que se desvían del uso tradicional del escritorio.

2004 - Digg se pone en Marcha

Se anunció que el sitio web "Digg" se pondría en marcha, propondría una forma innovadora de crear y encontrar contenido en internet mediante la promoción de noticias y enlaces democráticamente votado por los usuarios.

2004 - Facebook llega a la Web

Fue lanzado Facebook, que estaba entonces en su infancia, abierto sólo a los estudiantes. Con un millón de suscriptores justamente al final de 2004, Facebook se ha convertido en el medio de comunicación social más utilizado con más de 900 millones de usuarios a finales de 2004.

2005 - Lanzamiento de Youtube

En 2005, YouTube fue lanzado oficialmente, permitiendo a los usuarios compartir vídeos en línea. De un simple sitio para compartir vídeos en internet a la plataforma madura que se conoce hoy en día.

2006 - Lanzamiento de Twitter

Se puso en marcha en 2006. Con los años, la popularidad de Twitter ha aumentado de 1,6 millones de 'tweets' en 2007 con la impresionante cifra de 340 millones de dólares por día en marzo de 2012.

2006 - Web 3.0

Es una expresión que se utiliza para describir la evolución del uso y la interacción de las personas en internet a través de diferentes formas entre los que se incluyen la transformación de la red en una base de datos.

13 de diciembre de 2005 - Ruby on Rails

Es un framework de aplicaciones web de código abierto escrito en el lenguaje de programación Ruby, siguiendo el paradigma del patrón Modelo Vista Controlador, desarrollado por David Heinemeier Hansson.

2007 - iPhone llega al mercado

El año 2007 estuvo marcado por la aparición del iPhone, que fue sin duda responsable de la llegada de la nueva moda para las plataformas móviles y aplicaciones web. Ahora son accesibles por teléfono inteligente.

16 de abril de 2007 - Adobe Dreamweaver

Es una aplicación en programa de estudio (basada en la forma de estudio de Adobe Flash) que está destinada a la construcción, diseño y edición de sitios, vídeos y aplicaciones web basadas en estándares. Creado inicialmente por Macromedia.

2 de septiembre de 2008 - Google Chrome

Es un navegador web desarrollado por Google y compilado con base en componentes de código abierto como el motor de renderizado de WebKit y su estructura de desarrollo de aplicaciones.

22 de enero de 2009 - Mariadb

MariaDB es un sistema de gestión de bases de datos derivados de MySQL desarrollado por Michael Widenius.

2011 - Pedal de arranque

Para cerrar con esta línea del tiempo, a principios de 2011, la empresa Kickstarter, que facilita la financiación de proyectos en línea de forma participativa, ha llegado a los 4000 proyectos con más de 30 millones de dólares en donaciones. Por otra parte, casi el 44% de los proyectos se han iniciado con éxito desde esta plataforma.

2011 - Laravel

Laravel es un framework de código abierto para desarrollar aplicaciones y servicios web con PHP 5 y PHP 7, fue desarrollado por Taylor Otwell.

2011 - Oreja

Se creó como solución interna para solucionar las inconsistencias en el desarrollo dentro del equipo de ingeniería de Twitter. Básicamente, no se había establecido ninguna convención sobre las formas en las que los ingenieros de Twitter elegían cómo desarrollar la plataforma.

2013 - Explorador de Internet 11

Microsoft anuncia el último lanzamiento de este navegador.

2013 - Panorama

Microsoft anuncia que trasladará las cuentas de hotmail a Outlook.

2014 - HTML 5.0

Se publica oficialmente HTML 5.0.

2015 - Borde de Microsoft

Desarrollo para Microsoft, este navegador es el reemplazo de Internet Explorer.

2015 - AJAX

Acrónimo de Asynchronous JavaScript And XML (JavaScript asíncrono y XML), es una técnica de desarrollo web para crear aplicaciones interactivas.

3 de julio de 2018 - MaríaDB 10.3.8

Se anuncia una nueva versión de MariaDB, actualmente es su versión estable.
